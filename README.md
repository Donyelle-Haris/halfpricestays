# HalfPriceStays

HalfPriceStays — Experience luxury redefined. Bitcoin-exclusive bookings. 50% off all rooms.

A modern React application for booking luxury stays with cryptocurrency payments.

## Features

- **Bitcoin-exclusive bookings**: Secure cryptocurrency payments
- **50% off all rooms**: Exclusive discounts for crypto users  
- **Luxury stays**: Curated selection of high-end accommodations
- **Responsive design**: Built with Tailwind CSS for all devices
- **Fast performance**: Optimized with Vite build tool

## Tech Stack

- **React 18** - Frontend library
- **React Router DOM** - Client-side routing
- **Tailwind CSS** - Utility-first CSS framework
- **Vite** - Next-generation frontend tooling
- **PostCSS** - CSS processing
- **Cloudflare Workers** - Deployment platform

## Getting Started

### Prerequisites
- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/HalfPriceStay/HalfPriceStays.git
   cd HalfPriceStays
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.

## Deployment

This project is configured for deployment to Cloudflare Workers. See `wrangler.toml` for configuration.
